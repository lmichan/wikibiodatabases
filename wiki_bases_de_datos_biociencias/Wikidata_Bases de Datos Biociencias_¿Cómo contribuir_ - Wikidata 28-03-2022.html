¿Cómo contribuir?

  

Hemos documentado el proceso de generación y curación de entradas sobre bases de datos para biociencias de tal manera que sea fácil reproducirlo y sea normalizado.

Colaborar[editar | editar código]

Es recomendable antes de editar conocer los principios de Wikidata, su funcionamiento y hacer la visita guiada por la plataforma.

Antes de agregar nuevas entradas o editar la información por favor asegurate de que no estás generando duplicados, para esto hay que buscar el elemento en wikidata usando el nombre completo de la base de datos y la abreviatura, si no está en wikidata entonces se agrega, ver el procedimiento abajo.

Agregar nuevas bases de datos[editar | editar código]

Para colaborar en este Wikiproyecto si quieres agregar una nueva base de datos a wikidata puedes hacerlo de tres maneras distintas, las dos primeras sirven para agregar registros de uno en uno, la tercera permite agregar varios registros al mismo tiempo:

  1. Generar un nuevo elemento
  2. Usar el siguiente formulario para agregar información básica https://cradle.toolforge.org/#/subject/bioscience_database_corta
  3. Con Quick statements utilizando la siguiente plantilla básica
1. Un nuevo elemento[editar | editar código]
  • Agregar la etiqueta, se pone el nombre completo y la abreviatura en inglés
  • En alias se escribe la abreviatura y cualquier variable del nombre
  • En descripción con minúsculas se pone una descripción lo más detallada y breve posible
  • Página Web oficial
  • Instancia de base de datos biológica o médica según sea el caso
  • Fuente
2. Formulario[editar | editar código]

NOTA: En algunas ocasiones al usar el formulario se tarda o se traba la aplicación.

Utilizar este formulario para agregar nuevos registros con información básica https://cradle.toolforge.org/#/subject/bioscience_database_corta

Usar el siguiente formulario para agregar nuevos registros a wikidata con información más completa https://cradle.toolforge.org/#/subject/bioscience_database

En idioma generalmente usamos en para inglés y es para español

3. Quick statements[editar | editar código]
CREATE
LAST P31 Q4117139
LAST Len "Nombre de la base de datos"
LAST Den "Descripción de la base de datos"
LAST P856 "url"
LAST P1343 Elemento de la fuente por ejemplo Q110577062 (NAR dbs)

Idioma[editar | editar código]

Los registros de generan en inglés.

Dado que nuestro equipo habla español promovemos que además de generar el elemento en inglés, se edite también descripción en español.

Instancia de[editar | editar código]

Cada recurso debe corresponder a alguna de estas categorías:

https://www.wikidata.org/wiki/Q4117139 biological database - Wikidata
https://www.wikidata.org/wiki/Q1916557 medical database - Wikidata

Además puede ser de alguna de las siguientes subcategorías:

https://www.wikidata.org/wiki/Q5570651 global species database - Wikidata
https://www.wikidata.org/wiki/Q1789476 bibliographic database - Wikidata
https://www.wikidata.org/wiki/Q24577212 citizen science project - Wikidata

Web oficial[editar | editar código]

Al agregar la url se debe agregar también el idioma del recurso para evitar el mensaje de error que aparece con una bandera.
